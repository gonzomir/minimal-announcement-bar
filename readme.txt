== Description ==


== Installation ==

1. Extract the zip file and upload the folder `minimal-announcement-bar` to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add announcements from the Announcement menu in the WordPress admin.
